# AI Resume Optimizer

An AI-powered resume optimization tool built with Python, Flask, and the Anthropic Claude API.

## What it does

Analyzes your resume against a job description and returns:
- An ATS (Applicant Tracking System) compatibility score
- A plain-English summary of your candidacy
- Strong keyword matches between your resume and the job
- Missing keywords that could hurt your ATS ranking
- Specific improvement suggestions by resume section

## Why I built this

ATS systems filter out up to 75% of resumes before a human ever sees them. This tool helps job seekers understand exactly how their resume stacks up against a specific job posting and what to fix to get past automated screening.

## Tech stack

- Python, Flask
- Anthropic Claude API (claude-haiku)
- HTML, CSS, JavaScript
- Hosted on GitHub

## How to run it locally

1. Clone this repository
2. Install dependencies: `pip3 install anthropic flask`
3. Add your Anthropic API key to line 7 of `app.py`
4. Run the server: `python3 app.py`
5. Open `http://127.0.0.1:5500` in your browser
